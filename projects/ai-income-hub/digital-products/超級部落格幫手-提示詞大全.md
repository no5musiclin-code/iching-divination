## 1. 標題生成器

### 超級標題公式
```
你是一個專業的內容行銷專家，擅長創造高點閱率的文章標題。

請為主題：「[你的主題]」

生成 10 個標題，遵循以下公式：
- 數字型：數字 + 驚人好處
- 問題型：引發好奇心
- 比較型：VS 競品
- 承諾型：保證結果

每個標題要：
- 控制在 60 字以內
- 包含主要關鍵字
- 引發情緒反應

用以下格式回覆：
1. [標題]
```

### 病毒標題產生器
```
我需要一個會被瘋傳的標題。

主題：[你的主題]
目標讀者：[你的受眾]
主要好處：[他們會獲得什麼]

生成 5 個「點擊誘惑」標題，運用：
- 緊急感（FOMO）
- 獨家/秘密
- 數字量化
- 對比衝突

格式：
「[標題]」- [原因]
```

## 2. 文章架構師

### 完整文章大綱
```

目標字數：[1500/2000/3000] 字
目標讀者：[描述你的受眾]

請產生：
1. H1 標題

格式：

## 3. SEO 優化器

### Meta 描述產生器
```
為這篇文章產生 SEO 優化的 meta 描述：

標題：[你的標題]
主要關鍵字：[關鍵字]
目標受眾：[描述]

要求：
- 150-160 字元
- 包含主要關鍵字
- 包含次要關鍵字
- 引發點擊興趣
- 避免關鍵字堆疊

輸出格式：
描述：[你的描述]
```

### 關鍵字擴展器
```
我需要更多關鍵字來覆蓋「[主要關鍵字]」這個主題。

請產生：
1. 10 個長尾關鍵字（5-8字）
2. 5 個問題型關鍵字（以 how/what/why/when/where 開頭）
3. 5 個相關搜尋建議
4. 競爭度評估（高/中/低）

針對每個關鍵字，說明：
- 搜尋意圖
- 適合的內容類型
```

```

開場白類型（選擇一種或多種混合）：
A. 故事型：從個人經歷切入
B. 數據型：用統計數字抓住注意力
C. 問題型：直接提出讀者痛點

## 5. 呼籲行動產生器

### 超強 CTA
```
為這篇文章設計呼籲行動：

文章主題：[主題]
目標動作：[訂閱/購買/下載/分享/留言]

讀者現在的情緒：[興奮/困惑/猶豫/感動]

請產生：
1. 主 CTA 按鈕文字（10個選項）
2. 開場鉤子（引發行動的理由）
3. 潛在阻礙 + 消除方案
4. 緊迫感元素

格式：

原因：[為何有效]
```

### 軟性推銷文案
```
在文章「[標題]」中適當置入產品/服務推銷。

產品：[產品名稱]
優點：[3個主要優點]

請用「軟性」方式撰寫推銷段落：
- 不要直接說「買買買」
- 用故事、案例、證據說服
- 讓讀者自己想要

長度：200-300 字
```
