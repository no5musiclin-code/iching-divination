## 1. 開場白鉤子

### 爆款開場公式

```
你是一個社群媒體內容專家，擅長創造高互動的開場鉤子。

請為以下主題產生 10 個開場鉤子：

主題：[你的主題]
目標平台：[Instagram/Twitter/LinkedIn/Facebook]
目標受眾：[你的粉絲是誰]
情感目標：[激勵/震驚/溫馨/有趣/實用]

要求：
- 每個鉤子 15-30 字
- 能在 3 秒內抓住注意力
- 直接引發「想看下去」的衝動

格式：
1. [鉤子]
2. [鉤子]
...以此類推
```

### 不同類型的開場白

**問題型：**
```
產生 5 個「問句」開場白，主題是 [主題]

要求：
- 讓粉絲想要回答
- 營造參與感
- 可以是選擇題或問答題

格式：
「[問題]」
- 適用於：[平台]
```

**數字型：**
```
產生 5 個「數字列表」開場白，主題是 [主題]

要求：
- 數字要具體（不是「5個」而是「這5個」）
- 數字要大到讓人震驚
- 要符合主題

格式：
「[數字] [驚人宣言]」
```

**故事型：**
```
用「我曾經_____」或「我認識一個人_____」的格式
產生 3 個故事型開場白，主題是 [主題]

要求：
- 30 秒內可以讀完
- 有轉折或驚喜
- 能引發共鳴
```

## 2. 病毒標題產生器

### Twitter/X 病毒標題

```
我需要讓這篇內容在 X/Twitter 上瘋傳：

主題：[你的主題]
主要訊息：[你想傳達的重點]

請產生 10 個推文標題，運用以下技巧：

1. 懸念型：「這件事改變了我對___的看法」
2. 數字型：「我試了 30 天，結果___」
3. 衝突型：「大家都在說___，但其實___」
4. 實用型：「免費的___方法，99% 的人不知道」
5. 恐懼型：「如果你不做___，你會後悔」

每個標題要：
- 少於 280 字元
- 引發情緒反應
- 包含行動詞
```

### Instagram 標題產生器

```
為 Instagram 產生吸睛標題：

圖片/主題：[描述]
目標：[獲得讚/分享/留言/點擊連結]

產生 5 個標題，結合：
- 開場鉤子（引發好奇心）
- 主要內容
- Hashtags（最少 5 個）

格式：

Hashtags：[標籤1 #標籤2 #...]
```

## 4. Hashtag 生成器

### 超級 Hashtag 公式

```
我需要為這篇內容產生最佳 Hashtags：

主題：[主題]
內容類型：[教學/心得/新聞/娛樂/etc]
目標受眾：[描述]

請產生以下組合：

A. 高流量熱門標籤（10個）：
- 最多人搜尋的主題標籤

B. 中流量精準標籤（10個）：
- 相關但較小眾的標籤

C. 長尾利基標籤（10個）：
- 只有你的目標受眾會搜尋的標籤

D. 品牌標籤（2-3個）：
- 你的個人/品牌標籤

格式：
#標籤名（搜尋量估算）

注意：
- 總共 25-30 個標籤
- 混合大小標籤
- 避免過度使用的泛標籤
```

### Instagram Reels 專用標籤

```
為 Instagram Reels 影片產生標籤策略：

影片主題：[描述]
影片類型：[教程/段子/生活/vlog/etc]

產生：
1. 開頭鉤子用的標籤（3個）

5. 品牌標籤（2個）

總共：30 個標籤

格式：
#標籤 | 用途說明

## 5. CTA 呼籲行動

### 超強 CTA 模板

```
為這篇社群內容設計呼籲行動：

內容主題：[主題]
平台：[Instagram/Twitter/LinkedIn/etc]
目標行為：[留言/分享/點擊連結/投票/關注]

請產生：

A. 直接型 CTA（3個）：
- 直接要求讀者做某事
- 範例：「馬上點擊連結試試」

B. 問題型 CTA（3個）：
- 用問句引導行動
- 範例：「你同意嗎？留言告訴我」

C. 情境型 CTA（2個）：
- 設想讀者情境
- 範例：「如果你也曾經___，轉發給需要的人」

D. 獎勵型 CTA（2個）：
- 提供價值換取行動
- 範例：「回覆『秘技』，我私訊你完整版」

每個 CTA 要：
- 少於 20 字
- 具體明確
- 容易執行
```

### 多平台 CTA 適配

```
請將這個 CTA 改編到不同平台：

原始 CTA：[你的呼籲行動]
主要訊息：[核心價值]

改編版本：

1. Instagram 貼文：
「[CTA]」

2. Instagram Story：
「[短版 CTA，限時動態專用]」

3. Twitter/X：
「[140字內版本]」

4. LinkedIn：
「[專業語氣版本]」

5. Facebook：
「[較長說明版本]」

6. TikTok：
「[短版口語版本]」

## 6. 評論互動模板

### 引爆留言的問題

```
產生 10 個「讓粉絲非留言不可」的問題：

主題：[你的內容主題]
目標：[收集意見/增加互動/了解受眾/etc]

要求：
- 問題要具體，不要太開放
- 讓粉絲願意分享個人經歷
- 簡短好回覆
- 有爭議性的話題可以有，但不要引戰

格式：
1. [問題]
2. [問題]
...
```

### 回覆粉絲模板

```
當粉絲留言時，如何有溫度地回覆：

情境 A - 粉絲問問題：
「謝謝你的問題！關於___，我的建議是___。你還有其他想了解的嗎？」

情境 B - 粉絲分享個人經歷：
「謝謝你分享！___（共鳴或肯定）。這真的很不簡單！」

情境 C - 粉絲給予讚美：
「謝謝你！聽到你這樣說我很開心。如果你覺得有幫助，可以分享給需要的朋友喔！」

情境 D - 粉絲批評或抱怨：
「謝謝你的回饋（正面詮釋）。關於___，我會___。有其他建議也歡迎告訴我。」

情境 E - 引導更深層互動：
「這個話題很棒！想深入聊聊嗎？___（問一個更深入的問題）」
```

Week 1:
| 日期 | 類型 | 主題 | Hashtags |
| Day 1 | 教程 | ... | ... |
| Day 2 | 心得 | ... | ... |

（每週 5-7 篇內容，包括：
- 3 篇主要內容（教程/資訊）

- 混合不同內容類型
