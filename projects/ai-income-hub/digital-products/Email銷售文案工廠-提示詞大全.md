## 1. 開場白工廠

### 各種開場白類型

```
你是一個 Email 行銷專家，擅長寫高轉換率的銷售郵件。

請為以下情況產生開場白：

情況：[描述你的 Email 目的]
目標讀者：[你的受眾是誰]
主要優惠/訊息：[你想傳達的]

產生以下類型的開場白（每種 3 個）：

A. 故事型：
以一個小故事或個人經歷開始
格式：「我曾經___，直到有一天___」

B. 問題型：
用一個問題引發共鳴
格式：「你曾經___嗎？」

C. 數據型：
用統計數字抓住注意力
格式：「研究顯示___」

D. 直接型：
開門見山
格式：「我寫這封信是要告訴你___」

E. 引用型：
用名言或他人話語切入
格式：「___曾經說過：『___』」

每個開場白要：
- 少於 50 字
- 馬上引發興趣
- 與 Email 主題相關
```

### 不同溫度的開場白

```
根據 Email 類型，產生不同風格的開場白：

「[开场白]」

B. 溫暖型（已互動過）：
「[開場白]」
為什麼適用：有興趣，需要強化連結

C. 熱情型（現有客戶）：
「[開場白]」
為什麼適用：已有信任，可以直接推銷
```

## 3. Email 序列模板

### 歡迎序列（5 天）

```
設計一個 5 天的歡迎 Email 序列：

目標讀者：新訂閱用戶
最終目標：建立信任 → 介紹產品 → 促成第一次購買

Day 1 - 歡迎 Email：
主題：[建立第一印象]
內容要素：
- 謝謝訂閱
- 你是誰，為什麼寫這個
- 讀者能得到什麼
- CTA（連結資源或內容）

Day 2 - 價值展示：
主題：[展示你的專業]
內容要素：
- 一個實用的技巧或建議
- 具體的成果或改變
- CTA（閱讀某篇部落格文章）

Day 3 - 故事/背景：
主題：[建立情感連結]
內容要素：
- 你是如何開始的
- 你相信什麼
- 為什麼你做這件事
- CTA（閱讀「關於我」頁面）

Day 4 - 社會證明：
主題：[其他人怎麼說]
內容要素：
- 客戶見證或成功案例
- 媒體報導或獎項
- 你幫助過的人的數量
- CTA（連結案例研究）

Day 5 - 首次推銷：
主題：[介紹你的產品/服務]
內容要素：
- 過渡：「既然你已經認識我了...」
- 介紹你的主要產品
- 為什麼現在是最好時機
- 限時優惠或bonus
- CTA（直接購買或了解更多）

請為每一天產生完整的 Email 內容。
```

## 4. 銷售文案公式

### AIDA 公式

```
使用 AIDA 公式撰寫銷售 Email：

產品/服務：[描述]
目標讀者：[描述]
主要優點：[最重要的 3 個好處]

A - Attention（注意力）：
「[用一個大膽的宣言、數據或問題抓住注意力]」

I - Interest（興趣）：
「[用更具體的內容維持興趣，展示你理解讀者的問題]」

D - Desire（欲望）：
「[轉換興趣為欲望，強調如果問題解決了會有多美好]」

A - Action（行動）：
「[清楚告訴讀者下一步，現在就行動]」

每個部分要：
- 簡短有力
- 聚焦在讀者利益
- 為行動做好準備
```

### PAS 公式

```
使用 PAS 公式處理讀者的疑慮：

痛點問題：[讀者最大的困擾]

P - Problem（問題）：
「[描述問題如何影響讀者的生活或工作]」

A - Agitate（激化）：
「[讓問題看起來更嚴重，增加緊迫感]」

S - Solution（解決方案）：
「[介紹你的產品如何完美解決這個問題]」

格式：
[問題描述，100字以內]
↓
[激化描述，80字以內]
↓
[解決方案 + CTA，150字以內]
```

### FAB 公式

```
使用 FAB 介紹產品功能：

產品：[產品名稱]

F - Feature（功能）：
「[這個產品有什麼功能/特色]」

A - Advantage（優點）：
「[這個功能如何比競爭對手更好]」

B - Benefit（利益）：
「[這個優點對讀者有什麼具體好處]」

範例：
功能：這個軟體有 AI 自動寫作功能
優點：比手工寫作快 10 倍
利益：你每天可以節省 2 小時，用來陪伴家人或發展事業

請為你的 3 個主要功能/特色使用 FAB 公式。
```

## 5. 升級銷售序列

### 升級銷售模板

```
設計一個升級銷售（Upsell）序列：

原始產品：[產品名稱和價格]
升級產品：[產品名稱和價格]
升級理由：[為什麼客戶需要這個升級]

升級時機：客戶購買原始產品後立即顯示

主旨：「讓你的成果提升 2 倍：[升級產品]」
內容結構：

1. 恭喜購買（真誠）
2. 解釋為什麼寫這封信
3. 展示升級版的額外價值
   - 新功能帶來的好處
   - 與原產品如何搭配
4. 限時優惠（如果有的話）
5. 獨家升級價
6. 簡單的升級按鈕
7. 如果不需要也沒關係（消除壓力）

語氣：熱情但不要給壓力
重點：強調價值，不是推銷
```

### Cross-sell 交叉銷售模板

```
設計一個交叉銷售序列：

已購產品：[產品A]
推薦產品：[產品B]
關聯性：[為什麼這兩個產品相關]

主旨：「你可能也會喜歡：[產品B簡稱]」

內容要素：
1. 基於你的[產品A]，我推薦這個
2. [產品B] 如何補充[產品A]
3. 實際的好處（用 FAB 公式）
4. 客戶見證（如果有的話）
5. 限時 Bonus（如果有的話）
6. CTA 按鈕

語氣：像朋友推薦，不是在銷售
```

## 7. 數據優化指南

### 追蹤指標

```
設定你的 Email 行銷 KPI：

開信率（Open Rate）
- 行業平均：15-25%
- 你的目標：第一年 20%，第二年 25%

點擊率（Click Rate）
- 行業平均：2-5%
- 你的目標：第一年 3%，第二年 5%

轉換率（Conversion Rate）
- 產品頁面平均：1-3%
- 你的目標：2%

取消訂閱率（Unsubscribe Rate）
- 行業平均：0.5% 以下
- 超過 1% 就要檢查原因

每次 Email 需追蹤：
- 發送時間
- 開信率
- 點擊率
- 轉換率
- 回復（如果有的話）
```

## 使用說明

### Email 行銷工作流程

1. **策略階段**
   - 定義目標受眾
   - 設定你的 Email 目的
   - 選擇適用的模板

2. **內容創作**
   - 使用提示詞快速產出
   - 根據你的品牌聲音調整
   - 加入個人化元素

3. **發送優化**
   - 測試不同發送時間
   - 根據數據調整
   - 持續學習

4. **分析改進**
   - 每月檢視數據
   - 找出最好的 Email
   - 持續迭代

### 最佳實踐

- **個人化**：在 Subject 和開場白加入讀者名字
- **價值先行**：先給予，再索取
- **手機優先**：70% 的 Email 在手機上閱讀
- **持續測試**：永遠不要停止優化
- **真誠語氣**：沒有人喜歡被「賣」的感覺
